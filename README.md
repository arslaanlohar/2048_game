# 2048 Game 

A classic **2048 puzzle game** built from scratch using **Python** and **Tkinter**, featuring a sleek dark theme with a blue-toned tile palette.

## About

Slide and merge numbered tiles on a 4x4 grid to combine them and reach the elusive **2048** tile! Simple to play, hard to master.

##  Features

-  Custom **dark theme UI** with glowing blue tile gradients
-  Smooth controls — Arrow keys or WASD
-  Live **score** and **best score** tracking
-  Clear **Game Over** overlay when no moves are left
-  Win detection when you hit 2048 (keep playing for higher tiles!)
-  One-click **New Game** restart

##  Tech Stack

- **Python 3**
- **Tkinter** (built-in GUI library, no extra dependencies)

## 🚀 How to Run

```bash
git clone https://github.com/arslaanlohar/2048-game-tkinter.git
cd 2048-game-tkinter
python game_2048.py
```

No external packages required — just Python 3 with Tkinter (included by default in most Python installations).

## Controls

| Key | Action |
|-----|--------|
| ↑ / W | Move Up |
| ↓ / S | Move Down |
| ← / A | Move Left |
| → / D | Move Right |
